Welcome to FriendFork!

FriendFork is a social disaggregation service to make it easier to consume FriendFeed in Google Reader (It may be useful with other feed readers too, but Reader is the primary target). FriendFork makes several changes to the feeds produced by FriendFeed:
  • Produces multiple feeds, one for each friend group you specify (so you can, for example, "mark all as read" in one group without affecting others.
  • Hides Reader shared items that have no likes or comments. These items are replaced with a pointer to the user's Reader shared page so you can subscribe (these links show up only once per user).
  • Embeds YouTube videos so they can be played inline.
  • Minor formatting tweaks. For example, a timestamp is shown after each comment.

FriendFork is currently in an experimental phase of development. There are sure to be bugs, the interface is rather user-unfriendly, and there may be disruptive changes (such as a change to the feed urls used). Subscribe to the FriendFork blog to get announcements about any significant changes.
